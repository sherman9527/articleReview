## 二、政治合规类（最高优先级）

### 2.1 明令禁止词
| 禁用词 | 正确形式 | 法规依据 |
|--------|---------|---------|
| 满清 | 清朝 / 清代 | 国务院规定 |
| 蒙古大夫 | （禁用） | 民族歧视 |
| 支那 | 中国 | 歧视性历史词汇 |
| 八年抗战 | 十四年抗战 | 教育部2017年通知 |
| 南巡讲话 | 南方谈话 | 邓小平1992年南巡正式名称 |

### 2.2 领土主权敏感词
- **台湾**：不可与国家并列（不能写"台湾、日本、韩国"，应写"日本、韩国等国家和中国台湾地区"）
- **台湾法律** → 台湾地区有关规定
- **台湾国立大学** → 删去"国立"前缀（如"台湾清华大学"）
- **民国纪年**：1949年后不得使用，来自台湾资料须转换为公元纪年
- **香港/澳门**：不称"殖民地"，应称"英占时期""葡占时期"

### 2.3 非标准地理名称（须更正）
| 错误称谓 | 标准名称 |
|---------|---------|
| 斯普拉特利群岛 | 南沙群岛 |
| 埃佛勒斯峰 | 珠穆朗玛峰 |
| 东土耳其斯坦 | 我国新疆 |
| 阿鲁纳恰尔邦 | 我国藏南地区 |
| 西海 / 西海海域 | 黄海 / 黄海海域 |
| 南朝鲜 | 韩国 |
| 蒙古人民共和国 | 蒙古国（1992年改名） |
| 汉城 | 首尔（2005年改名） |
| 福摩萨 / 福尔摩萨 | 台湾 |

### 2.4 引用领导人讲话的准确性
> 引用党和国家领导人的讲话、文件，**必须逐字准确**，不得删减或改动措辞。

典型差错：
- 十二大：邓小平提出"建设**有中国特色的**社会主义"（含"有"和"的"字）
- 十七大：胡锦涛报告为"节约**能源**资源和保护**生态**环境"（不可删"能源"和"生态"）
- 习近平讲话完整题目：《**在文艺工作座谈会上的讲话**》（不可简化为《文艺讲话》）
- 核心价值观：是"**法治**"不是"法制"（两字含义不同）

### 3.2 常见别字（音同形近）
- 副（量词）vs 幅（宽度）：一**副**手套、一**幅**画
- 灸（艾灸）vs 炙（烧烤）：针**灸**、**炙**手可热
- 股（大腿）vs 骨（骨头）：悬梁刺**股**（非刺骨）
- 籍（籍贯）vs 藉（垫底/践踏）：原**籍**、杯盘狼**藉**
- 阕（词的量词）vs 阙（宫殿建筑）：一**阕**词、宫**阙**
- 蛰伏 vs 蜇伏：**蛰**伏（冬眠）、**蜇**（蜂蜇）

## 五、语法差错类

### 5.1 主谓搭配
- 数量/数字：应"增加/减少"，不能"提高/降低"
- 注视（用于人/物）vs 倾听（用于声音）：不可混用
- 崭新的文物：矛盾（文物必为旧物）
- 四大发明之一 vs 四大发明（整体）

### 5.2 语义重复
- 凯旋归来 → 凯旋（已含归来）
- 忍俊不禁地笑 → 忍俊不禁（已含笑）
- 时速50公里/小时 → 时速50公里（"时速"已含"小时"）
- 由于…的原因 → 由于…
- 原因是因为 → 原因是 / 因为
- 是否…与否 → 择一

### 5.3 标点差错（高频）
- **一逗到底**：长段落应在每个完整句子后用句号，不能全用逗号
- **标题加句号**：标题末尾通常不加句号
- **省略号前的叹号不可省略**：`！……` 不可写作 `……`
- **书名号使用**：书名、篇名必须用《》，不能用引号代替
- **法律名称**：全称加书名号《中华人民共和国刑法》，简称不加（刑法）

## 六、数字用法规范（GB/T 15835-2011）

| 场景 | 规范 | 错误示例 | 正确示例 |
|------|------|---------|---------|
| 公历世纪 | 阿拉伯数字 | 二十一世纪 | 21世纪 |
| 年份 | 阿拉伯数字 | 二〇二四年 | 2024年 |
| 百分数 | 数字+% | 百分之七十五 | 75% |
| 相邻概数 | 中间不加顿号 | 三、四天 | 三四天 |
| 量符号 | 按国标 | 时速50km/h | 速度50km/h |
| 量名称 | 规范量名 | 瓦数40W | 功率40W |

## 七、引文格式规范（GB/T 7714-2015）

| 文献类型 | 标识 | 格式模板 |
|---------|------|---------|
| 专著 | [M] | 作者. 书名[M]. 出版地: 出版社, 年份: 页码. |
| 期刊 | [J] | 作者. 题名[J]. 刊名, 年份, 卷(期): 页码. |
| 学位论文 | [D] | 作者. 题名[D]. 学校, 年份. |
| 报纸 | [N] | 作者. 题名[N]. 报纸名, 日期(版次). |
| 网络资源 | [EB/OL] | 作者. 题名[EB/OL]. URL, 发布日期[引用日期]. |

**高频引文差错**：
1. 缺文献类型标识（[M][J]等）
2. 多作者超3人未用"等"
3. 网络文献未注明访问日期
4. 书名用引号而非书名号
5. 起止页码格式不统一（应用"–"连接）

## 八、出版质量标准

- **合格标准**：每万字差错不超过1个（学术书籍）
- **严重差错**：政治性错误、常识性错误、知识性错误（每个计3个差错）
- **特别严重**：违反宪法、损害国家尊严等 → 直接判定不合格
- **检查范围**：封面、书脊、版权页、目录、正文、插图表格、参考文献、索引
